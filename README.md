# suri-ai
News verification and summarization front-end (ChatGPT-like UI).

- One input field for news link or topic.
- URL mode: fetches article text and outputs summary + fake/real likelihood.
- Topic mode: returns curated verified news links.
- No OpenAI key required.
- `resources.html` holds courses and verified news list.
- `signup.html` for user registration with validation.
- `login.html` for user login with simulation.

Features:
- Typing-animated response.
- Responsive design.
- Simple heuristic legitimacy check.
- Stop/cancel analysis.
- Sign-up and login forms with client-side validation.
